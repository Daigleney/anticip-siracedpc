La clé API transite de votre navigateur vers votre Cloudflare Worker (HTTPS), qui la relaie vers Anthropic. Elle n'est jamais stockée sur un tiers. Si vous cochez « mémoriser », clé et URL sont conservées dans ce navigateur uniquement.

Le Worker est nécessaire car les navigateurs bloquent les appels directs vers des APIs tierces (politique CORS), même depuis GitHub Pages.
